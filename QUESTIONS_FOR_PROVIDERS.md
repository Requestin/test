# Вопросы для провайдеров GPU-серверов

## Цель использования
Планирую запускать модель **FlashVSR** ([GitHub](https://github.com/OpenImagingLab/FlashVSR)) для видео супер-разрешения. Модель требует сборки библиотеки **Block-Sparse-Attention** с компиляцией CUDA-кода.

## Чек-лист критичных требований

**Обязательно:**
- [ ] **Полная изоляция GPU** — вся мощность GPU закреплена только за мной (не shared)
- [ ] NVIDIA A100 или A800 с 40+ ГБ памяти (желательно 80 ГБ)
- [ ] Ubuntu Linux с root/sudo доступом
- [ ] CUDA Toolkit 12 или возможность установки
- [ ] Python 3.11.13 или возможность установки
- [ ] Conda/Miniconda
- [ ] Компиляторы (gcc, g++, CMake, Ninja)
- [ ] Git и Git LFS
- [ ] Доступ в интернет (GitHub, Hugging Face)
- [ ] Достаточно RAM для компиляции CUDA-кода (рекомендуется 16+ ГБ)

**Желательно:**
- [ ] Предустановленное ML-окружение
- [ ] Готовые образы с PyTorch и CUDA
- [ ] Техническая поддержка с опытом работы с ML
- [ ] Возможность тестового доступа
